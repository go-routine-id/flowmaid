One engine, four faces

The same crate powers everything — every feature lands everywhere at once.

📦 Embed it

cargo add flowmaid — render_svg(text) for one-shot SVG, or the scene/route API for interactive canvases with live edge re-routing.

🌐 Web

Compiles to wasm32 out of the box. The playground ships the whole engine in a 166 KB bundle — mermaid.js is ~2.5 MB.

🖥️ Desktop

flowmaid-desktop: a native editor with live text, draggable nodes, zoom/pan, and real file workflow.

⌨️ CLI

flowmaid diagram.mmd -o out.svg — pipe-friendly, line-numbered errors, ~60 ms end-to-end on 5,000-node graphs.

Mermaid parity roadmap

Goal: mermaid.js functionality, pure-Rust edition. Full board with issues in #10 — contributions welcome.

  • flowchart — shapes, links, cycles v0.1
  • erDiagram — crow's foot, entity tables v0.2
  • style / classDef / ::: custom colors v0.4
  • subgraph — nesting, titles, direction v0.5
  • Interactive drag API (scene / route)
  • Fan-out A --> B & C, more link types
  • classDiagram · sequenceDiagram · stateDiagram
  • journey · pie · mindmap
  • $$math$$ in labels (KaTeX subset)
  • gantt · gitGraph · timeline · the whole catalog
